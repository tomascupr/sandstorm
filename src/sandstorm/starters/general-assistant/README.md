# General Assistant

General-purpose starter for research, document work, support operations, and software tasks.

## Run it

```bash
ds "Compare Notion, Coda, and Slite for async product teams"
```

## Example prompts

- Compare three competitors and write a short brief for leadership.
- Summarize these call notes and extract follow-up actions.
- Triage these support notes into priorities and next steps.
- Crawl a docs site and draft a short API summary.
- Review this repo for security risks and quick wins.

## How to customize

1. Edit `system_prompt_append` in `sandstorm.json` with your team context, tone, or constraints.
2. Replace the example prompts with the workflows you want teammates to run most often.
3. Only rewrite the base `system_prompt` if you want to change the starter's overall behavior.
