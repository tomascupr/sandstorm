# Research Brief

Starter for fast research briefs, competitive scans, and vendor comparisons.

## Run it

```bash
ds "Compare Linear, Jira, and Asana for a 50-person product org"
```

## Example prompts

- Compare three vendors and recommend which one we should pilot first.
- Research the current landscape for AI support tooling and summarize the tradeoffs.
- Review a market segment and identify the most credible competitors.
- Compare a new policy proposal across public source material and write a short brief.

## How to customize

1. Put your company context, evaluation criteria, or house style into `system_prompt_append`.
2. Update the prompt examples so the starter mirrors your real decision workflows.
3. Edit the base `system_prompt` only if you need a different research posture or output style.
